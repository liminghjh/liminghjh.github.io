文件包含

  • include 和request两种包含

语法：include “1.php” 相当于在此处替换1.php的内容
request “1.php”
区别：include报错级别低，后续代码可以进行，request报错级别高，后续代码不能执行；

  • 路径

绝对路径:C:/windows/index.php
相对路径: ./ 代表当前文件夹开始 ../代表上一级文件夹开始
include “index.php”相当于 include “./index.php” 属于相对路径
